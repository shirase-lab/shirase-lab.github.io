使い方ガイド

うちわ作りからコンビニで実寸プリントするまでの流れと、よくある「〇〇したい」をまとめました。

印刷までの流れ

はじめてでも、うちわが完成します

  1. パーツを置く

    ツールバーの「スタンプ」「画像」「ペン」などでうちわに要素を配置。文字はキャンバスの空いた所をタップで追加できます。「テンプレ」パネルから定番デザインを選んでもOK。

  2. 色・装飾を整える

    要素をタップして選び、下の編集パネル（「サイズ」／「フォント」または「色」／「加工」）で色・大きさ・フォント・縁取り・効果（ぷくぷく／きらきら／ネオン）を調整。「色」ツールの推しカラーは、ドラッグ＆ドロップやタップで配色できます。

  3. 仕上がりを確認

    ツールバーの「3D」で、持ち手付きの立体イメージで仕上がりを確認できます。

  4. コンビニ印刷データを作る

    ツールバーの「印刷」→「コンビニ印刷データ作成」で、A3 高解像度の PDF を書き出します。

  5. 印刷する

    共有メニューからコンビニのプリントアプリ（セブン-イレブンの netprint など）へ送るか、保存して持ち込み。家庭用プリンタは「印刷」ボタンから直接印刷できます（大きいうちわは A4×2 枚の分割プリセットも）。

Q&A

こんなときは（〇〇したい）

タップでひらく、よくある質問

Q文字を入れたい+
キャンバスの何もない所をタップすると入力欄が開き、文字を追加できます（要素を選んでいるときは、いったん空いた所をタップして選択を外すと追加できます）。置いた文字をタップ → 下の編集パネルで文言・フォント・サイズ・色を変更できます（文字をダブルタップでも編集できます）。
Q色を変えたい／推しカラーにしたい+
要素を選んで出る編集パネルの色から変更できます。ツールバーの「色」では推しカラー（おすすめ配色）を選べ、色を要素へドラッグ＆ドロップ、またはタップで配色できます。同じ色をまとめて変えることもできます。
Q画像を入れたい+
ツールバーの「画像」→「取込」で、端末の画像を取り込めます（一度取り込んだ画像はサムネイル履歴から再利用できます）。取り込むと背景削除の画面が自動で開きます。※公式画像・ロゴの無断使用は著作権・肖像権の侵害になり得ます。文字＋フリー素材中心の制作をおすすめします。
Q画像の背景を消したい+
画像を取り込むと背景削除の画面が自動で開きます（あとから直すときは画像を選び「加工」の「背景を編集／背景削除を編集」から）。モードは3つ＝「自動」（被写体をAIで自動抽出）／「色キー」（背景の色を画像でタップし、「しきい値」で消える範囲を調整）／「領域」（消したい場所をタップし、そこからつながった部分だけを消す。複数か所OK）。やり直すときは「リセット」、決めて取り込むときは「取込」。グラデーションや影のある背景はきれいに抜けないことがあります。
QSVG・スタンプに縁取り（フチ）を付けたい+
SVG／スタンプを選んで出る編集パネルの「加工」の「縁取り」で、各パスにフチを付けられます（「フチを追加」で外→内に多重も可。ぷくぷく・きらきら・ネオンと併用できます）。※文字そのものへのフチ付けは、現在は提供していません。
Qぷくぷく／きらきら／ネオンにしたい+
文字や要素を選んで、編集パネルの「加工」から選択します。きらきら・ぷくぷくは背景にも追加できます（ツールバーの「うちわ」パネルの「背景キラキラ」「背景ぷくぷく」から）。ネオンは文字・要素向けの効果です。
Q連うちわ（複数ページ）について+
複数ページ（連うちわ）機能は終了し、いまは1枚のうちわ作りに専念しています。複数のうちわは、それぞれ別ファイルとして作成・印刷してください。以前の複数ページファイルを開いた場合は1枚目だけが読み込まれます（必要なら、開いたときに各ページを別ファイルへ分割して保存できます）。
Qうちわの形（シルエット）を変えたい+
ツールバーの「うちわ」を開き、「うちわテンプレート」のサムネから形を選べます。「うちわ」パネルの「新規作成」時にも選べます。
Q保存・読み込みしたい+
ツールバーの「うちわ」を開き、「保存」（プロジェクト .uchiwa か 画像 PNG を選択）・「開く」（.uchiwa を読み込み）・「共有」ができます。編集内容は自動保存され、次回は前回の続きから復元されます。
Qテンプレ・定番フレーズを使いたい+
「テンプレ」パネルから定番デザインや、多言語のファンサ定番フレーズを挿入できます。
Q制作の過程を動画にしたい+
ツールバーの「録画」で制作過程をタイムラプス録画し、MP4 として保存・共有できます。

ご利用上の注意

  • 公式画像・ロゴの無断使用は著作権・肖像権の侵害になり得ます。文字＋フリー素材中心の制作をおすすめします。
  • 会場によっては写真付き・大型ボードの持ち込みが禁止されています。各会場のルールをご確認ください。
  • ジャンボうちわのサイズ（約28.5×29.5cm）が会場持ち込み上限の目安です。
